Conceitos para Criação


O nosso site é feito em flask python, o flask é uma biblioteca do python onde você consegue criar um website a partir do python.

Além disso temos uma combinação de HTML, CSS e JavaScript em implementação do metodo mobile-first onde estruturamos um site que suporte diversas telas, assim tendo também uma opção mais pratica de navbar para mobile e desktop, que permite que o usuário navegue entre diferentes páginas do site.

A parte de bd é feita com o sqlalchemy nos 3 arquivos criacao.py, tabelas.py e consultas.py.

O criacao.py gera uma conexão com o banco de acordo com o SGBD escolhido, no caso o postgresql
O tabelas.py é quem cria as tabelas usando classes e as popula com uma sessão criada, atualizando os dados do real banco de dados.
O consultas.py como o nome diz com ele que podemos fazer SELECT, DELETE, UPDATE, INSERT etc.
